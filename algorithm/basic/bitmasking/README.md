# 비트 연산
&, |, ^, ~

* & : 둘 다 1일때 -> 1 그 외는 0
* | : 적어도 하나가 1일 때 -> 1, (0 | 0) 만 0
* ^ : 둘다 같으면 0, 다르면 1
* ~ : 비트 반전


# 시프트 연산
비트의 위치를 이동시키는 시프트 연산자  
시프트 연산자는 << 또는 >> 다음에 비트를 이동 시킬 횟수를 지정함  
<<는 비트를 왼쪽으로 이동시키고, >>는 비트를 오른쪽으로 이동시킨  

비트를 오른쪽으로 이동시켰을 때 1이 들어갈 공간이 없다면 1은 사라짐
계속 비트를 오른쪽으로 이동시키면 1은 모두 사라지고 최종 결과는 0b0이 됨
